Bài 1. Nhập bán kính r của đường tròn, tính chu vi diện tích

Code html:

<html>
<head>
<title>Chương trình tính diện tích và chu vi hình tròn</title>
<meta http-equiv="Content-type"content="text/html; charset=utf-8">
<script type="text/javascript">function tinhCV(r)
{
if (r>0)
chuVi.value=2 * r * Math.PI;
else
alert("Bán kính không hợp lệ");
}
function tinhDT(r)
{
if (r>0)
dienTich.value=Math.PI * r * r;
}
function reset()
{
dienTich.value=chuVi.value=banKinh.value="";
}</script>
</head>
<body>
<p>Nhập vào bán kính: <input type="text" id="banKinh"/> (đơn vị)</p>
<p>Chu vi là: <input type="text" id="chuVi" disabled="true" /> (đơn vị)</p>
<p>Diện tích là: <input type="text" id="dienTich" disabled="true" /> (đơn vị<sup>2</sup>)</p>
<input type="button" value="Tính chu vi" onclick="tinhCV(banKinh.value);"/>
<input type="button" value="Tính diện tích" onclick="tinhDT(banKinh.value);"/>
<input type="button" value="Nhập lại" onclick="reset();"/>
</body>
</html>

Bài 7. Giải phương trình ax2+bx=c=0

Code html:

<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<title>Giải phương trình bậc hai</title>
<script type="text/javascript">
function giaiPTb2(a,b,c)
{
if (a==0)
if(b==0)
if (c==0)
alert("Vô nghiệm");
else
alert("Vô số nghiệm");
else
x1.value = -c/b;
else
{
delta=b * b - (4 * a * c);
if(delta <0)
alert("Vô nghiệm");
else
{
if(delta ==0)
x1.value=x2.value=-b/2/a;
else
{
x1.value=(-b-Math.sqrt(delta))/2/a;
x2.value=(-b+Math.sqrt(delta))/2/a;
}
}
}
}</script>
</head>
<body>
<h3>Phương trình có dạng ax<sup>2</sup>+bx+c=0</h3>
<p>Nhập vào hệ số a: <input type="text" id="a"/></p>
<p>Nhập vào hệ số b: <input type="text" id="b"/></p>
<p>Nhập vào hệ số c: <input type="text" id="c"/></p>
<p>x1=<input type="text"id="x1" disabled="true"/></p>
<p>x2=<input type="text" id="x2" disabled="true"/></p>

Bài 8. Tìm min max của một dãy số

Code html:

<html>
<head>
<meta http-equiv="Content-type" content="text/html; charset=utf-8">
<title>Tìm min max của dãy số</title>
<script type="text/javascript">function timMinMax(ds)
{
n=ds.length;
for(i=0;i<n;i++)
{
ds=ds.replace(","," ");
}
for(i=0;i<n;i++)
{
ds=ds.replace(" "," ");
}
txtDaySo.value=ds;
var list=ds.split(" ")
min=Number(list[0]);
for (i=0;i<list.length;i++)
{
if (min>Number(list[i]))
min=Number(list[i]);
}
txtMin.value=min;
max=Number(list[0]);
for (i=0;i<list.length;i++)
{
if (max<Number(list[i]))
max=Number(list[i]);
}
txtMax.value=max;
}</script>
</head>
<body>
<p>Nhập vào 1 dãy số: <input type="text" id="txtDaySo"/> (Các số phân tách nhau bởi dấu phẩy, hoặc dấu cách)</p>
<p>Chữ số nhỏ nhất là: <input type="text" id="txtMin" disabled="true"/></p>
<p>Chữ số lớn nhất là: <input type="text" id="txtMax" disabled="true"/></p>
<input type="button" value="Tìm" onclick="timMinMax(txtDaySo.value);"/>
</body>
</html>
